#### CST-003 | Code | 풀페이지 팝업의 헤드라인/서브카피/이미지/CTA를 서버에서 교체할 수 있는 구조가 필요 — 필수 (Builder 결정 항목)

**상황:** 풀페이지 팝업의 헤드라인/서브카피/이미지/CTA를 서버에서 교체할 수 있어야 합니다. PO가 현재 서버 교체가 가능하다고 확인했습니다.

**처리하지 않으면:** 서버 교체 불가 시 매번 앱 업데이트가 필요하여 메시지 테스트 반복이 어려움

**Builder가 결정할 사항:** 기존 서버 교체 구조를 활용하여 새 메시지 콘텐츠를 배포

**이 작업 관점에서의 판단:** 기존 구조 재사용이므로 추가 개발 없음. 되돌림 비용 낮음.

#### CST-001 | Experience | 팝업의 "월 4만원대" 표시와 실제 결제 금액이 세그먼트/쿠폰에 따라 다를 수 있어 사용자 혼란 가능 — 권장

**상황:** 팝업 CTA에 "4만원대"로 표시하는데, LOW/MEDIUM 세그먼트의 쿠폰 금액이 다릅니다. PO가 Exploration에서 모든 세그먼트에서 최저가 수강권 + 쿠폰 적용 시 4만원대가 맞다고 확인했습니다. 이용약관에 가격 표시 관련 제한 조항은 없습니다.

**처리하지 않으면:** 팝업에서 본 가격과 랜딩페이지 가격이 다르면 신뢰 저하 및 이탈 가능

**선택지:**

| 선택 | 이점 | 내용 | 리스크 | 되돌림 비용 |
|------|------|------|--------|------------|
| inject — 현재 "4만원대" 표시 유지 | PO 확인 완료. 모든 세그먼트에서 사실. | 팝업에 "4만원대" 그대로 노출. 랜딩페이지에서 정확한 금액 확인. | 세그먼트별 미세 차이가 있을 수 있으나, "4만원대"는 범위 표현으로 커버됨. | 낮음 — 서버에서 카피 교체 가능 |
